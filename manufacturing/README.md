# Manufacturing Output Package

This directory contains the complete supplied Altium manufacturing/output archive for **RPI GPIO BOARD - V2.0**, generated on **January 30, 2025**.

## Contents

- Top and bottom copper Gerbers
- Top and bottom solder-mask Gerbers
- Top and bottom silkscreen/legend Gerbers
- Top and bottom paste layers
- Board profile and mechanical layers
- Assembly, courtyard, component-outline, component-center, pad-master, and 3D-body layers
- NC drill files and drill reports
- Aperture, Gerber-generation, extension, layer-pair, DRC-rule, and status reports
- CAMtastic review files

## Important distinction

These are manufacturing and review outputs. They are not the editable Altium source project. The original `.PrjPcb`, `.PcbDoc`, `.SchDoc`, BOM, and pick-and-place files were not included in the supplied archive.

Use the original Gerber and drill files for CAM review. The images in `assets/` are documentation visuals only.
